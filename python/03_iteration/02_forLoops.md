## Iteration For Loops

* Structures for iterating through iterables

Syntax

```python
for item in iterable:
    codeblock

for variable in iterable:
    codeblock
```

### For Loop Example

```python
mylist = ['a','b','c']
for ch in mylist:
    print(ch)
```
